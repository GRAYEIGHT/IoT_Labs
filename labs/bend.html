Sense Bend

In this workshop, you will connect a flex sensor to an Arduino analog pin
and program the Arduino to light an LED when the potentiometer reaches a defined threshold.

Introduction

Flex sensors measure bend. When you flex the sensor, the resistance across the sensor increases. The resistance of the flex sensor changes when the metal pads are on the outside of the bend (text on inside of bend).

To measure the value of the flex sensor, it will be connected to a voltage divider circuit. This circuit acts like a potentiometer. The circuits outer laeds are connected to +5volts and ground (GND). The middle lead is connected to an Arduino analog pin. As the flex sensor bends, its resistance value will change and cause the the voltage at the middle lead to change, just like a potentiometer.

For this lab, you will connect a flex sensor to an Arduino Uno analog input. When the flex sensor value reaches a certain level, as defined by the program sketch, the LED will light. At the same time, the program will send flex sensor data to the computer through the USB connector.

To begin, grab the parts listed in the Parts section and assemble them as shown in the diagram in the Breadboard section.

Once your circuit is assembled, use the Arduino IDE to type (or cut and paste) the code in the Program section and upload it to your Arduino. To do this, plug the Arduino board into your USB port. Then select the proper port in Tools > Serial Port > (the comm port of your Arduino).

Next, upload the program by going to File > Upload or typing ctrl+U. After the program loads, feel free to knock over your project!

Parts

The following are the parts for this lab.

One (1) LED. Your kit contains a red, green and yellow LED, use whichever color you like.

The longest lead is positive.

Two (1) 1K ohm resistors. The first resistor limits the current flowing to the LED. This makes sure the LED is not damaged. The second resistor is used to pull up the output of the motion sesnor to +5volts.

One (1) flex sensor. When you bend the sensor, the LED will light.

Wiring

Program

/*
	Sense Flex
*/

#define THRESHOLD  512;
int flexPin = 2;
int ledPin = 3;

// The setup routine runs once when you upload a program or press the reset button
void setup()
{
	Serial.begin(9600);
}

// The loop routine runs over and over, forever.
}
void loop() 
{
	int flexValue = analogRead(flexPin);
	
	if (flexValue >= THRESHOLD) // turn LED on:
	{
			Serial.println(flexValue);
    		digitalWrite(ledPin, HIGH);
	}
 	else {						// turn LED off:
    		digitalWrite(ledPin, LOW);
  	}
	
	delay(1000); 
  	Serial.println(flexValue);
}

Not Working?

Program not uploaded

Make sure the proper serial port is selected. Select the serial port in the Arduino IDE menu tools > serial port.

LED not lighting up

LEDs will only work in one direction. Try turning it around. (no need to worry, installing it backwards does no permanently harm the LED)

Going Further

Reverse the LED Lighting Order

Instead of having the LED light when a magnet is near, have is normally lit and turn it off when the sensor senses a magnetic field.

Change the program as follows:

if (flexVal == LOW) {		// turn LED on:
	digitalWrite(ledPin, HIGH);
}

Then reload the program by clicking the upload button of selecting ctrl-u.

Use two LEDs to flex Status

Add a second LED that indicates when their is no magnetic near. When there is no magnetic field, the first LED will light, when a magnetic field is detected, the first LED will cut off and the second LED will light.

Add Sound

Use a second LED to indicate another threshold level. The first level can act as a “warning” message and second level can act as “danger” level. Use a yellow LED for warning and a red LED for danger.
