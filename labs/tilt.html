Sense tilt

In this workshop, you will connect a tilt sensor to an Arduino digital pin
and program the Arduino to light an LED when the sensor tips over.

Introduction

Tilt sensors tell the orientation of an object. Is it standing straight up or did it tip over? With a tilt sensor, you can detect when something is moved, tilted, or shaken.

Just like a switch, tile sensors are either open or closed. They behave like a switch, when standing up they are open, when tilted they are closed.

For this lab, you will connect a tilt sensor to the Arduino Uno. The output of the sensor is fed into a digital pin, which is configured in the Setup() function to act as an input. When tilted the Arduino Uno will light an LED to indicate that the sensor is tilted (or closed). When you tilt the sensor upright, the LED will turn off.

A resistor, called a pull-up resistor, is connected to the output of the tilt sensor. This resistor "pulls" the digital input pin to a value of one (1) when the button is not pushed.

To begin, grab the parts listed in the Parts section and assemble them as shown in the diagram in the Breadboard section.

Once your circuit is assembled, use the Arduino IDE to type (or cut and paste) the code in the Program section and upload it to your Arduino. To do this, plug the Arduino board into your USB port. Then select the proper port in Tools > Serial Port > (the comm port of your Arduino).

Next, upload the program by going to File > Upload or typing ctrl+U. After the program loads, feel free to knock over your project!

Parts

The following are the parts for this lab.

One (1) LED. Your kit contains a red, green and yellow LED, use whichever color you like.

The longest lead is positive.

Two (1) 1K ohm resistors. The first resistor limits the current flowing to the LED. This makes sure the LED is not damaged. The second resistor is used to pull up the output of the button to +5volts.

One (1) tilt sensor. When you tilt your circuit, the LED will light.

Wiring

Program

/*
Sense Tilt
Turn on an LED when a tilt sensor is tipped. LED is normally off.
*/

int tiltPin = 2;
int ledPin = 3;
 
// The setup routine runs once when you upload a program or press the reset button
void setup()
{
	// Sets the digital pin as output
	pinMode(ledPin, OUTPUT);
	pinMode(tiltPin, INPUT);
}

// The loop routine runs over and over, forever.
}
void loop() 
{
	int tiltState = digitalRead(buttonPin);
	
	if (tiltState == HIGH) 
	{						// turn LED on:
		digitalWrite(ledPin, HIGH);
 	}
 	else 
	{						// turn LED off:
		digitalWrite(ledPin, LOW);
	}

	delay(1000);					// Delay for 1 second (1000 milliseconds)
}

Not Working?

Program not uploaded

Make sure the proper serial port is selected. Select the serial port in the Arduino IDE menu tools > serial port.

LED not lighting up

LEDs will only work in one direction. Try turning it around. (no need to worry, installing it backwards does no permanently harm the LED)

Tilt sensor not seated

The tilt sensor may not be seated properly in the breadboard. If this occurs, it will not make a connection to the jumper wires. Check to make sure that the sensor is pushed firmly into the breadboard.
