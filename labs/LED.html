Light an LED

In this workshop, you will connect an LED to an Arduino digital pin and program
the Arduino to make the LED blink on and off in one second intervals.

Introduction

An LED (light emitting diode) is a component that lights up when current passes through it. LED's come in a range of colors but a singel LED’s typically only emits a single color. But, there are LED’s that can emit a range of colors. These LED's have multiple leads, one for each color they support. LED's have polarity, which means they have a positive (+) lead and a negative (-) lead. If not properly connected, an LED will not light. Some uses of LED’s are as indicator lights or backlighting for panels.

For this lab you will use your Arduino Uno to turn on and off an LED every second, which makes the LED appear to be blinking.

To begin, grab the parts listed in the Parts section and assemble them as shown in the diagram in the Breadboard section.

Once your circuit is assembled, use the Arduino IDE to type (or cut and paste) the code in the Program section and upload it to your Arduino. To do this, plug the Arduino board into your USB port. Then select the proper port in Tools > Serial Port > (the comm port of your Arduino).

Next, upload the program by going to File > Upload to I/O Board (ctrl+U). Finally, bask in the glory that your blinking LED offers.

Parts

The following are the parts for this lab.

One (1) LED. Your kit contains a red, green and yellow LED, use whichever color you like.

The longest lead is positive.

One (1) 1K ohm resistor. The resistor limits the current flowing to the LED. This makes sure the LED is not damaged.

Wiring

Program

/*
	Light an LED
	Repeatedly turns on and off an LED once a second, then off for one second.
*/

int ledPin = 2;

// The setup routine runs when you upload a program or press the reset button.
void setup()
{
	pinMode(ledPin, OUTPUT);		// Sets the digital pin to output mode
}

// The loop routine runs over and over, forever.
void loop()
{
	digitalWrite(ledPin, HIGH);		// turn the LED on (high voltage level)
	delay(1000);					// wait for one (1) second
	digitalWrite(ledPin, LOW);		// turn the LED off (low voltage level)
	delay(1000);					// wait for one (1) second
}

Not Working?

Program not uploaded

Make sure the proper serial port is selected. Select the serial port in the Arduino IDE menu tools > serial port.

LED not lighting up

LEDs will only work in one direction. Try turning it around. (no need to worry, installing it backwards does no permanently harm the LED)

Going Further

Change the Pin

The LED is connected to pin two (2) of the Arduino. You can use any of the Arduino’s digital pins. To use a different digital pin, unplug the jumper from pin two (2) and move it to a pin of your choice (from 0- 13).

Change the program as follows:

from: int ledPin = 2;
to:   int ledPin = (new pin);

Then reload the program by clicking the upload button of selecting ctrl-u.

Change the Blink Time

Want your LED to blink faster or slower?

Change the program as follows:

from: delay(1000);					// wait for one (1) second
to:   delay(new time on);
from: delay(1000);					// wait for one (1) second
to:   delay(new time off);
