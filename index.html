I2C Sensors

Inter-Integrted Circuit (I2C) sensors use a three-wire communication protocol to communicate their data. The I2C interface can be used to write configuraion parameters to the sensor as well as read its data. We will learn to use the Arduino's I2C interface pins to read I2C sensors and to report their status via its serial interface.
